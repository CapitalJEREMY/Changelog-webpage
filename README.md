# Rejuvinated

Simple static website project containing the main landing page and changelog.

## Files

- `index.html` - main homepage
- `changelog.css` - styles for the changelog or related content
- `Rejuvenated.css` - site stylesheet
- `Rejuvenated.html` - alternate or additional page

## Usage

Open `index.html` in a web browser to view the site.

## Notes

This repository is initialized with Git in the project root.
